# Challenge Data Science - Alura

Este proyecto forma parte del Challenge de Data Science de Alura Latam.

## Objetivo
Analizar los datos de ventas de cuatro tiendas para identificar:
- Facturación total
- Categorías más vendidas
- Productos más vendidos
- Promedio de evaluación de clientes
- Costo promedio de envío

## Tecnologías utilizadas
- Python
- Pandas
- Jupyter Notebook

## Archivos
- challenge-alura.ipynb → análisis de datos de las tiendas
